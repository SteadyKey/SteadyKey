ダウンロード

以下のボタンを押すと、リリースアーカイブを新しいタブで開きます。ダウンロードはブラウザのダウンロード機能で行ってください。

直接ダウンロードリンク（新しいタブで開く）

※ ZIP をダウンロード後、展開して Chrome の「拡張機能」から読み込んでください。詳しい手順は下の「インストール手順」をご覧ください。

インストール手順

  1. ダウンロードボタンで ZIP を取得し、任意のフォルダに展開してください。
  2. Chrome を開き、chrome://extensions/ にアクセスします。
  3. 右上の「デベロッパーモード」を有効にし、「パッケージ化されていない拡張機能を読み込む」をクリック。
  4. 展開したフォルダ内の dist フォルダ（または展開先のルート）を選択して読み込みます。
  5. 拡張機能アイコンをクリックして SteadyKey を起動し、初回は Machine Key が生成されます。
注意：拡張機能をアンインストールすると Machine Key が消え、保存済みデータは復号不能になります。バックアップが必要な場合はエクスポート機能を利用してください（将来的な機能）。
